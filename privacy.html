Privacy Policy

Data Collection

This extension collects and processes the following data:

  • Your OpenAI API key (stored locally in Chrome's sync storage)
  • Database schema information (table and column names) from your Metabase instance
  • Natural language queries you enter to generate SQL

Data Usage

The collected data is used exclusively to:

  • Generate SQL queries based on your natural language input
  • Provide relevant schema context to the AI model
  • Improve the accuracy of SQL generation

Data Storage

All data is stored locally in your browser:

  • API keys are stored in Chrome's sync storage
  • Database schema is cached locally for 24 hours
  • No data is stored on external servers except for the necessary API calls to OpenAI

Third-Party Services

This extension uses the following third-party services:

  • OpenAI API for SQL generation
  • Your Metabase instance for schema information

Contact

For any privacy-related questions, please contact the extension developer.
